Welcome to AI Draft for Google Docs™

Thank you for installing the add-on. Follow the steps below to begin using AI Draft inside Google Docs™.

1. Open the Sidebar

Open any Google Docs™ document, then go to:
Extensions → AI Draft for Google Docs™ → Open Sidebar

2. Enter Your API Key

AI Draft uses a “bring your own key” approach. You may use:

  • Your Google Gemini API key
  • Your OpenAI API key

Your API key is stored only inside your Google account’s Apps Script property storage. It is never collected or shared with us.

How to get an API key

Quick guidance:

  • Google Gemini API key: Available in your Google AI Studio settings.
  • OpenAI API key: Available in your OpenAI account dashboard.

Each provider manages its own usage and billing. Please review your provider’s policies if you have questions about usage.

3. Start Using AI Draft

You can now use AI tools directly inside Google Docs™:

  • Generate text from a prompt
  • Rewrite or rephrase paragraphs
  • Summarize selected text
  • Expand or shorten content
  • Support for multiple languages
