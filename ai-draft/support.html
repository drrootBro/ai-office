Support

Having trouble with RootBro AI-Draft? Start with the common issues below. If you still need help, email us and we’ll do our best to get you unstuck.

Contact email: support@rootbro.com
Please include your Word version, operating system, and a short description (or screenshot) of the issue.

Common issues & quick checks

AI-Draft sidebar does not appear

Make sure the add-in is installed and enabled in Word:

  • Go to Insert → Add-ins or Insert → My Add-ins.
  • Confirm that RootBro AI-Draft is listed and enabled.
  • If needed, remove and re-add the add-in from the manifest file or Office Store.
Buttons don’t respond when I click “Generate”

Please check the following:

  • Ensure your internet connection is working.
  • Confirm that your API keys are valid and not expired in the Settings tab.
  • Try switching to another model provider (e.g. OpenAI → Gemini) to see if the issue is provider-specific.
License not recognized after purchase

If you bought a lifetime license but AI-Draft still shows you as on a trial:

  • Make sure you are using the same email address you used during purchase.
  • In the AI-Draft sidebar, open the License section and click “Check license” again.
  • If the issue persists, contact support@rootbro.com with your receipt from Lemon Squeezy.
API key invalid or not working

If you see errors about invalid or missing API keys:

  • Double-check that the key was copied correctly.
  • Confirm the key has the correct permissions in your provider dashboard (OpenAI, Google, Anthropic, etc.).
  • Make sure you are selecting a model that exists for your account.

When you contact support

To help us help you faster, please include:

  • Your operating system (Windows / macOS / Web / Other).
  • Your Microsoft Word version (e.g. Word for Microsoft 365).
  • A short description of what you were doing.
  • Any error messages visible in the add-in.
  • (Optional) A screenshot of the AI-Draft sidebar and your document.
