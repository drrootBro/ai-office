Install & setup guide

Follow the steps below to install RootBro AI-Draft in Microsoft Word and connect your own AI API keys.

Tip: This page is safe to share with colleagues or students. It does not expose any of your license information.

1. Install the add-in in Word

Option A · From a manifest file (side-load)
  1. Download the latest manifest.xml file from the official RootBro AI-Draft repository.
  2. Open Microsoft Word, go to Insert → Add-ins → My Add-ins.
  3. Choose Upload My Add-in (or a similar option) and select the manifest.xml file.
  4. Once installed, open AI-Draft from the Home or Insert tab to show the sidebar.

Exact menu names may vary slightly between Word for Windows, Mac, and Word Online.

Option B · From the Office Store (when available)
  1. In Word, open Insert → Add-ins → Store (or Get Add-ins).
  2. Search for "RootBro AI-Draft".
  3. Click Add to install the add-in.
  4. Launch the add-in and pin the sidebar for quick access while writing.

This option will be available after AI-Draft is approved in the Office Add-ins marketplace.

2. Activate your lifetime license

If you purchased a lifetime license through Lemon Squeezy, you will receive an email with your receipt and license information.

  1. Open the AI-Draft sidebar in Word and navigate to the License or Upgrade section.
  2. Click "Check license". AI-Draft will contact the RootBro license server and verify your email/license status.
  3. Once verified, premium features will be unlocked on this device.

If you run into any issues during license activation, please visit the Support page or contact us using the email listed there.

3. Add your API keys (BYOK)

RootBro AI-Draft follows a Bring Your Own Keys model: you connect your own AI providers, and the add-in simply offers a clean interface inside Word.

3.1 Open the API settings

  1. In the AI-Draft sidebar, open the Settings tab.
  2. Locate the section labeled API keys or Model providers.

3.2 Add specific providers

OpenAI
  1. Log in to your OpenAI account and generate an API key from the dashboard.
  2. Copy the key and paste it into the OpenAI field in AI-Draft.
  3. Save the settings.
Google Gemini
  1. Log in to your Google Cloud or Gemini API console and create an API key.
  2. Paste the key into the Gemini field in AI-Draft.
  3. Save the settings.
Anthropic Claude
  1. Create an API key in your Anthropic account.
  2. Paste the key into the Claude field in AI-Draft.
  3. Save the settings.

AI-Draft never shares or resells your keys. They are used only to call the providers you configure, from Word, under your account.

4. First use: Draft, rewrite, and summarize

Draft new content
  1. Place your cursor where you want new content in your document.
  2. In the AI-Draft sidebar, choose Draft and describe what you want (e.g. “Intro paragraph for a project report”).
  3. Click Generate, review the result, then click Insert to add it to your document.
Rewrite existing content
  1. Select a paragraph in your Word document.
  2. Choose Rewrite in the AI-Draft sidebar.
  3. Optionally specify the goal (e.g. “more concise”, “more formal”, “simpler English”).
  4. Generate and insert the improved version.
Summarize long text
  1. Select the section you want summarized.
  2. Choose Summarize in AI-Draft.
  3. Pick the style you want (short paragraph, bullet points, executive summary).
  4. Insert the summary into your document or notes.

5. Troubleshooting

If AI-Draft is not appearing, not activating, or not calling your AI models correctly, please see the Support page for common solutions and contact information.
